# Fractal example

- 3.3 core profile
- prereqs - conan, cmake
- deps - glfw, glew, imgui, glm, stb, assimp, fmt

# How to build and run:

```shell
./build.sh && ./run.sh
```

# Controls

- `mouse drag` - rotate camera against (0, 0, 0)
- `space + mouse drag` - rotate camera against current position
- `wasd` - move camera
- `q` - quit
